## Overview
GigFlow is a web application designed to streamline the process of applying for gigs and managing user profiles. It provides a user-friendly interface for both applicants and administrators.

## Features
- **User Authentication**: Secure login and registration for users.
- **Gig Management**: Create, view, and apply for gigs.
- **User Profiles**: Manage user information and application history.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/gigflow.git
   cd gigflow
   ```
2. **Install dependencies**:
   Navigate to both the `client` and `server` directories and run:
   ```bash
   pnpm install
   ```
3. **Run the application**:
   - For the client:
     ```bash
     cd client
     pnpm run dev
     ```
   - For the server:
     ```bash
     cd server
     node index.js
     ```
4. **Open your browser** and navigate to `http://localhost:3000` to view the application.
